# Telemetria do Antigravity

Esta pasta contém os arquivos de telemetria gerados durante a execução dos workflows.

## Arquivos:

- `current_execution.json` - Execução atual em andamento
- `metrics.json` - Métricas consolidadas
- `files_tracker.txt` - Lista de arquivos criados
- `commands_tracker.txt` - Comandos executados
- `duration.txt` - Duração da execução
- `validation.log` - Logs de validação
- `archive/` - Execuções anteriores (histórico)

## Não commitar:

Os arquivos de telemetria não devem ser commitados (estão no .gitignore).
Apenas a estrutura (README.md e .gitkeep) deve ser versionada.
