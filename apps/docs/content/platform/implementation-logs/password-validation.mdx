# Password Validation Implementation

## Overview

Production-ready password validation system with 5 security requirements, real-time strength indicator, and comprehensive client-side validation.

## Components

### PasswordValidator Component

**Location:** `apps/admin/components/password-validator.tsx`

**Features:**

- Real-time validation of 5 security requirements
- Visual strength indicator (5 levels: Very Weak â†’ Strong)
- Check/X icons for each requirement
- Shows only when field is focused or has errors
- Full accessibility (ARIA labels, roles)
- Dark mode support

**Requirements Validated:**

1. Minimum 8 characters
2. At least one lowercase letter
3. At least one uppercase letter
4. At least one number
5. At least one special character

**Exports:**

```typescript
export function PasswordValidator({ password, className });
export function validatePasswordRequirements(password: string);
export const PASSWORD_REGEX;
export const PASSWORD_ERROR_MESSAGE;
```

## Integration

### Form Schema (Zod)

```typescript
password: z.string()
  .min(8, 'Password must be at least 8 characters')
  .regex(PASSWORD_REGEX, PASSWORD_ERROR_MESSAGE);
```

### Form Implementation

```tsx
// State for focus tracking
const [isPasswordFocused, setIsPasswordFocused] = useState(false);

// Password input with focus handlers
<Input
  {...register('password')}
  type={showPassword ? 'text' : 'password'}
  onFocus={() => setIsPasswordFocused(true)}
  onBlur={(e) => {
    setIsPasswordFocused(false);
    register('password').onBlur(e);
  }}
/>;

// Conditional validator display
{
  (isPasswordFocused || errors.password) && (
    <PasswordValidator password={watch('password') || ''} />
  );
}
```

## Validation Mode

**Mode:** `onChange` for instant feedback

```typescript
useForm({
  mode: 'onChange',
  reValidateMode: 'onChange',
});
```

## Visual Feedback

All form fields show visual validation:

- **Green border:** Valid and touched
- **Red border:** Invalid and touched
- **Default:** Not yet validated

## Security Regex

```regex
^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[^A-Za-z0-9])[A-Za-z\d!@#$%^&*()_+\-=\[\]{};':"\\|,.<>\/?~`]+$
```
