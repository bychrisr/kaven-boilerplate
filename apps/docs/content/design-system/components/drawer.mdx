# Drawer

> Painel lateral que desliza sobre o conteúdo para exibir navegação ou informações adicionais.

## Variações

### Básico

```tsx
import {
  Drawer,
  DrawerContent,
  DrawerHeader,
  DrawerTitle,
} from '@/components/ui/drawer';

function BasicDrawer() {
  const [open, setOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setOpen(true)}>Abrir Drawer</Button>

      <Drawer open={open} onOpenChange={setOpen}>
        <DrawerContent>
          <DrawerHeader>
            <DrawerTitle>Título do Drawer</DrawerTitle>
          </DrawerHeader>
          <div className="p-4">Conteúdo do drawer</div>
        </DrawerContent>
      </Drawer>
    </>
  );
}
```

### Posições

```tsx
// Left (padrão)
<Drawer side="left">...</Drawer>

// Right
<Drawer side="right">...</Drawer>

// Top
<Drawer side="top">...</Drawer>

// Bottom
<Drawer side="bottom">...</Drawer>
```

## API

| Prop           | Tipo                                     | Padrão   | Descrição             |
| -------------- | ---------------------------------------- | -------- | --------------------- |
| `open`         | `boolean`                                | -        | Controla visibilidade |
| `onOpenChange` | `(open: boolean) => void`                | -        | Callback quando muda  |
| `side`         | `'left' \| 'right' \| 'top' \| 'bottom'` | `'left'` | Posição do drawer     |
