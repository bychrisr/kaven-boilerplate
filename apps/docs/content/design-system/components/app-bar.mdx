# AppBar

Barra de navegação superior com suporte a logo, menu e ações.

## Uso Básico

import { Button } from '../../../components/ui/button';
import { Avatar, AvatarFallback } from '../../../components/ui/avatar';

## Posições

```tsx
<AppBar position="fixed">Fixo (padrão)</AppBar>
<AppBar position="sticky">Sticky</AppBar>
<AppBar position="static">Static</AppBar>
```

## Boas Práticas

### ✅ Fazer

- Manter altura consistente (64px)
- Logo à esquerda
- Ações à direita

### ❌ Não Fazer

- AppBar muito alto
- Muitos itens de navegação
- Sem responsividade mobile
