# Button

> Componente de botão completo com suporte a loading states, ícones, e todas as variações semânticas do Kaven Design System.

## Variações

### Contained (Filled)

Botões com background sólido e sombra z8.

```tsx
import { Button } from '@/components/ui/button';

<Button variant="contained" color="primary">Primary</Button>
<Button variant="contained" color="secondary">Secondary</Button>
<Button variant="contained" color="success">Success</Button>
<Button variant="contained" color="warning">Warning</Button>
<Button variant="contained" color="error">Error</Button>
<Button variant="contained" color="info">Info</Button>
```

### Outlined

Botões com borda e background transparente.

## Com Ícones

```tsx
import { Plus, Trash2, Download } from 'lucide-react';

<Button variant="contained" color="primary">
  <Plus />
  Novo Usuário
</Button>

<Button variant="outlined" color="error">
  <Trash2 />
  Excluir
</Button>

// Icon only
<Button variant="contained" color="primary" size="icon">
  <Plus />
</Button>
```

## API

| Prop        | Tipo                                                                      | Padrão        | Descrição                         |
| ----------- | ------------------------------------------------------------------------- | ------------- | --------------------------------- |
| `variant`   | `'contained' \| 'outlined' \| 'text'`                                     | `'contained'` | Estilo do botão                   |
| `color`     | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'primary'`   | Cor semântica                     |
| `size`      | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| 'icon'`                          | `'md'`        | Tamanho                           |
| `loading`   | `boolean`                                                                 | `false`       | Estado de carregamento            |
| `disabled`  | `boolean`                                                                 | `false`       | Desabilita o botão                |
| `fullWidth` | `boolean`                                                                 | `false`       | Largura total                     |
| `asChild`   | `boolean`                                                                 | `false`       | Renderiza como child (Radix Slot) |
