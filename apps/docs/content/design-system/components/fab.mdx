# FAB (Floating Action Button)

> Botão circular flutuante para ação principal da tela.

## Básico

```tsx
import { FAB } from '@/components/ui/fab';
import { Plus } from 'lucide-react';

<FAB>
  <Plus />
</FAB>;
```

## Posições

```tsx
// Bottom Right (padrão)
<FAB className="fixed bottom-6 right-6">
  <Plus />
</FAB>

// Bottom Left
<FAB className="fixed bottom-6 left-6">
  <Plus />
</FAB>
```
