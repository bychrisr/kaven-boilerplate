# Card

> Superfície elevada para agrupar informações relacionadas.

## Exemplo Básico

```tsx
import { Card } from '@/components/ui/card';

<Card className="p-6">
  <h3 className="text-lg font-bold mb-2">Título do Card</h3>
  <p className="text-muted-foreground">
    Conteúdo do card com informações relevantes.
  </p>
</Card>;
```

## Variantes

```tsx
// Outlined
<Card className="border-2">...</Card>

// Elevated (com sombra)
<Card className="shadow-xl">...</Card>

// Flat (sem borda)
<Card className="border-none shadow-none">...</Card>
```

## API

| Prop        | Tipo        | Padrão | Descrição              |
| ----------- | ----------- | ------ | ---------------------- |

| `children`  | `ReactNode` | -      | Conteúdo do card       |
