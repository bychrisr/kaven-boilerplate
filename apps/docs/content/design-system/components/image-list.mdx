# Image List

> Componente para exibir coleções de imagens em grid.

## Básico

```tsx
import { ImageList, ImageListItem } from '@/components/ui/image-list';

<ImageList cols={3} gap={8}>
  <ImageListItem>
    <img src="/image1.jpg" alt="Image 1" />
  </ImageListItem>
  <ImageListItem>
    <img src="/image2.jpg" alt="Image 2" />
  </ImageListItem>
  <ImageListItem>
    <img src="/image3.jpg" alt="Image 3" />
  </ImageListItem>
</ImageList>;
```

| Prop   | Tipo     | Padrão | Descrição         |
| ------ | -------- | ------ | ----------------- |
| `cols` | `number` | `3`    | Número de colunas |
| `gap`  | `number` | `8`    | Espaçamento       |
