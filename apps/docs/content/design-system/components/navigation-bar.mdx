## API

| Prop       | Tipo                      | Padrão      | Descrição                                        |
| ---------- | ------------------------- | ----------- | ------------------------------------------------ |
| `position` | `'top' \| 'bottom'`       | `'top'`     | Posição fixa ou relativa                         |
| `variant`  | `'default' \| 'elevated'` | `'default'` | Estilo visual                                    |
| `logo`     | `ReactNode`               | -           | Elemento de logo                                 |
| `items`    | `NavigationItem[]`        | `[]`        | Lista de links de navegação                      |
| `actions`  | `ReactNode`               | -           | Elementos à direita (desktop) ou abaixo (mobile) |

### NavigationItem Object

```typescript
{
  label: string;
  href?: string;
  icon?: ReactNode;
  badge?: number;
  active?: boolean;
  onClick?: () => void;
}
```
