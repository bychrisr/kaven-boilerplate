# Toggle Button

> Botões que podem ser alternados entre estados ativo/inativo.

## Básico

```tsx
import { ToggleButton } from '@/components/ui/toggle-button';

<ToggleButton value="bold" aria-label="Bold">
  <Bold className="h-4 w-4" />
</ToggleButton>;
```

## Grupo

```tsx
import { ToggleButtonGroup } from '@/components/ui/toggle-button';

<ToggleButtonGroup type="single" value={alignment} onValueChange={setAlignment}>
  <ToggleButton value="left">
    <AlignLeft className="h-4 w-4" />
  </ToggleButton>
  <ToggleButton value="center">
    <AlignCenter className="h-4 w-4" />
  </ToggleButton>
  <ToggleButton value="right">
    <AlignRight className="h-4 w-4" />
  </ToggleButton>
</ToggleButtonGroup>;
```

## API

| Prop              | Tipo                         | Padrão | Descrição            |
| ----------------- | ---------------------------- | ------ | -------------------- |
| `value`           | `string`                     | -      | Valor do botão       |
| `pressed`         | `boolean`                    | -      | Estado pressionado   |
| `onPressedChange` | `(pressed: boolean) => void` | -      | Callback quando muda |
