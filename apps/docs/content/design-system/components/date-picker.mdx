# Date Picker

> Componente para seleção de datas com interface de calendário.

## Básico

```tsx
import { DatePicker } from '@/components/ui/date-picker';

function MyForm() {
  const [date, setDate] = useState<Date>();

  return (
    <DatePicker
      selected={date}
      onSelect={setDate}
      placeholder="Selecione uma data"
    />
  );
}
```

## Com Restrições

```tsx
// Data mínima
<DatePicker
  selected={date}
  onSelect={setDate}
  minDate={new Date()}
  placeholder="Data futura"
/>

// Data máxima
<DatePicker
  selected={date}
  onSelect={setDate}
  maxDate={new Date()}
  placeholder="Data passada"
/>

// Dias desabilitados
<DatePicker
  selected={date}
  onSelect={setDate}
  disabled={(date) => date.getDay() === 0 || date.getDay() === 6}
  placeholder="Apenas dias úteis"
/>
```

## Formatos


## API

| Prop          | Tipo                                | Padrão                 | Descrição                     |
| ------------- | ----------------------------------- | ---------------------- | ----------------------------- |
| `selected`    | `Date \| undefined`                 | -                      | Data selecionada              |
| `onSelect`    | `(date: Date \| undefined) => void` | -                      | Callback quando muda          |
| `mode`        | `'single' \| 'range' \| 'multiple'` | `'single'`             | Modo de seleção               |
| `minDate`     | `Date`                              | -                      | Data mínima permitida         |
| `maxDate`     | `Date`                              | -                      | Data máxima permitida         |
| `disabled`    | `(date: Date) => boolean`           | -                      | Função para desabilitar datas |
| `placeholder` | `string`                            | `'Selecione uma data'` | Texto placeholder             |
| `dateFormat`  | `string`                            | `'dd/MM/yyyy'`         | Formato de exibição           |
