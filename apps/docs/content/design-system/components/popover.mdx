# Popover

> Overlay flutuante para exibir conteúdo adicional ao clicar.

## Básico

```tsx
import {
  Popover,
  PopoverTrigger,
  PopoverContent,
} from '@/components/ui/popover';

<Popover>
  <PopoverTrigger asChild>
    <Button variant="outlined">Abrir Popover</Button>
  </PopoverTrigger>
  <PopoverContent>
    <div className="space-y-2">
      <h4 className="font-semibold">Título</h4>
      <p className="text-sm">Conteúdo do popover.</p>
    </div>
  </PopoverContent>
</Popover>;
```

## API

| Prop           | Tipo                      | Padrão | Descrição             |
| -------------- | ------------------------- | ------ | --------------------- |
| `open`         | `boolean`                 | -      | Controla visibilidade |
| `onOpenChange` | `(open: boolean) => void` | -      | Callback quando muda  |
