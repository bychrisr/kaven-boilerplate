## Collapse

Útil para expandir/colapsar seções de conteúdo verticalmente.

<ComponentPreview>
  <div className="flex flex-col items-center gap-4 w-full max-w-sm">
    {() => {
      const [checked, setChecked] = React.useState(false);
      return (
        <>
          <Button onClick={() => setChecked(!checked)} fullWidth>
            {checked ? 'Hide Details' : 'Show Details'}
          </Button>
          <Collapse in={checked}>
            <Paper className="p-4 mt-2 bg-gray-50">
              Lorem ipsum dolor sit amet, consectetur adipiscing elit.
              Suspendisse malesuada lacus ex, sit amet blandit leo lobortis
              eget.
            </Paper>
          </Collapse>
        </>
      );
    }}
  </div>
</ComponentPreview>

```tsx
<Collapse in={checked}>
  <div className="p-4 mb-2">Conteúdo colapsável...</div>
</Collapse>
```

## API

Todas as transições aceitam as seguintes props:

| Prop            | Tipo      | Padrão  | Descrição                                         |
| --------------- | --------- | ------- | ------------------------------------------------- |
| `in`            | `boolean` | -       | Se `true`, o componente transiciona para entrada. |
| `timeout`       | `number`  | `300`   | Duração da animação em ms.                        |
| `unmountOnExit` | `boolean` | `false` | Se `true`, desmonta o filho ao sair.              |

### Slide Props

| Prop        | Tipo                                  | Padrão   | Descrição           |
| ----------- | ------------------------------------- | -------- | ------------------- |
| `direction` | `'up' \| 'down' \| 'left' \| 'right'` | `'down'` | Direção da entrada. |
