## API

Todas as transições aceitam as seguintes props:

| Prop            | Tipo      | Padrão  | Descrição                                         |
| --------------- | --------- | ------- | ------------------------------------------------- |
| `in`            | `boolean` | -       | Se `true`, o componente transiciona para entrada. |
| `timeout`       | `number`  | `300`   | Duração da animação em ms.                        |
| `unmountOnExit` | `boolean` | `false` | Se `true`, desmonta o filho ao sair.              |

### Slide Props

| Prop        | Tipo                                  | Padrão   | Descrição           |
| ----------- | ------------------------------------- | -------- | ------------------- |
| `direction` | `'up' \| 'down' \| 'left' \| 'right'` | `'down'` | Direção da entrada. |
