# Stat Card

> Cards especializados para exibir métricas e KPIs com visualização de tendências e mini-gráficos integrados. Componente signature do Dashboard Kaven.

## Quando Usar

- **Dashboards**: Exibir métricas principais (usuários, receita, conversões)
- **Relatórios**: Apresentar KPIs com comparação temporal
- **Analytics**: Mostrar dados com contexto visual imediato
- **Painéis Executivos**: Resumir informações críticas de negócio

## Cores Semânticas

| Cor         | Classe do Ícone    | Classe do Background | Uso                                   |
| ----------- | ------------------ | -------------------- | ------------------------------------- |
| **Primary** | `text-primary`     | `bg-primary/10`      | Métricas principais (usuários, total) |
| **Blue**    | `text-blue-500`    | `bg-blue-500/10`     | Receita, vendas, conversões           |
| **Yellow**  | `text-yellow-500`  | `bg-yellow-500/10`   | Alertas, pendências, taxas            |
| **Green**   | `text-green-500`   | `bg-green-500/10`    | Sucesso, crescimento, aprovações      |
| **Red**     | `text-destructive` | `bg-destructive/10`  | Erros, cancelamentos, problemas       |

## API / Propriedades

### StatCard Component (Proposta)

| Propriedade  | Tipo                                                  | Padrão             | Descrição                                    |
| ------------ | ----------------------------------------------------- | ------------------ | -------------------------------------------- |
| `title`      | `string`                                              | -                  | Título do card (ex: "Total de Usuários")     |
| `value`      | `string \| number`                                    | -                  | Valor principal a ser exibido                |
| `icon`       | `LucideIcon`                                          | -                  | Ícone do Lucide React                        |
| `trend`      | `number`                                              | `0`                | Percentual de mudança (positivo ou negativo) |
| `trendLabel` | `string`                                              | `"últimos 7 dias"` | Label descritivo da tendência                |
| `color`      | `'primary' \| 'blue' \| 'yellow' \| 'green' \| 'red'` | `'primary'`        | Cor semântica do card                        |
| `chartData`  | `Array<{v: number}>`                                  | -                  | Dados para mini-gráfico (opcional)           |
| `className`  | `string`                                              | -                  | Classes CSS adicionais                       |
| `loading`    | `boolean`                                             | `false`            | Estado de carregamento                       |

### Exemplo de Uso (Componente Proposto)

```tsx
import { StatCard } from '@/components/ui/stat-card';
import { Users } from 'lucide-react';

<StatCard
  title="Total de Usuários"
  value={2456}
  icon={Users}
  trend={12}
  trendLabel="últimos 7 dias"
  color="primary"
  chartData={[{ v: 20 }, { v: 40 }, { v: 30 }, { v: 70 }, { v: 50 }]}
/>;
```

## Melhores Práticas

### ✅ Fazer

- **Use ícones semânticos** que representem claramente a métrica
- **Formate números** com `toLocaleString()` para melhor legibilidade
- **Mostre contexto temporal** (últimos 7 dias, vs. mês anterior)
- **Use cores consistentes** para tipos de métricas similares
- **Implemente loading states** para melhor UX
- **Torne responsivo** com classes `md:` e `lg:`
- **Adicione mini-gráficos** para métricas com histórico relevante

### ❌ Não Fazer

- ❌ **Não use cores aleatórias** - siga a paleta semântica
- ❌ **Não omita unidades** - sempre mostre % ou R$ quando aplicável
- ❌ **Não exiba valores sem contexto** - sempre mostre a tendência
- ❌ **Não use ícones genéricos** - cada métrica deve ter ícone específico
- ❌ **Não ignore estados de erro** - mostre feedback quando dados falharem
- ❌ **Não sobrecarregue com informação** - mantenha foco no valor principal
