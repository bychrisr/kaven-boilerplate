# Breadcrumbs

> Componente de navegação que mostra a hierarquia de páginas até a localização atual.

## Básico

```tsx
import { Breadcrumbs, BreadcrumbItem } from '@/components/breadcrumbs';

<Breadcrumbs>
  <BreadcrumbItem>
    <Link href="/dashboard">Dashboard</Link>
  </BreadcrumbItem>
  <BreadcrumbItem>
    <Link href="/users">Usuários</Link>
  </BreadcrumbItem>
  <BreadcrumbItem current>Lista</BreadcrumbItem>
</Breadcrumbs>;
```

## API

| Prop        | Tipo                  | Padrão | Descrição             |
| ----------- | --------------------- | ------ | --------------------- |
| `separator` | `string \| ReactNode` | `'/'`  | Separador entre items |
| `children`  | `ReactNode`           | -      | BreadcrumbItems       |

### BreadcrumbItem Props

| Prop       | Tipo        | Padrão  | Descrição         |
| ---------- | ----------- | ------- | ----------------- |
| `current`  | `boolean`   | `false` | Se é o item atual |
| `children` | `ReactNode` | -       | Conteúdo do item  |
