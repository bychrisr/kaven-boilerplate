# Mega Menu

> Menu dropdown grande com layout em colunas para navegação complexa.

```tsx
<MegaMenu>
  <div className="grid grid-cols-3 gap-6 p-6">
    <div>
      <h3 className="font-semibold mb-2">Categoria 1</h3>
      <ul className="space-y-1">
        <li>
          <Link href="#">Item 1</Link>
        </li>
        <li>
          <Link href="#">Item 2</Link>
        </li>
      </ul>
    </div>
    <div>
      <h3 className="font-semibold mb-2">Categoria 2</h3>
      <ul className="space-y-1">
        <li>
          <Link href="#">Item 3</Link>
        </li>
        <li>
          <Link href="#">Item 4</Link>
        </li>
      </ul>
    </div>
  </div>
</MegaMenu>;
```
