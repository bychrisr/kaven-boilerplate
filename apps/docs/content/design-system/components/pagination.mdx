# Pagination

> Componente para navegar entre múltiplas páginas de conteúdo.

## Exemplo Básico

```tsx
import { Pagination } from '@/components/ui/pagination';

<Pagination count={10} page={page} onChange={(newPage) => setPage(newPage)} />;
```

## Com Data Table

```tsx
const [page, setPage] = useState(0);
const [rowsPerPage, setRowsPerPage] = useState(10);

const { data } = useUsers({
  page: page + 1,
  limit: rowsPerPage,
});

const totalPages = Math.ceil((data?.pagination.total ?? 0) / rowsPerPage);

<div className="p-4 border-t border-border/50 flex items-center justify-between">
  <div className="text-sm text-muted-foreground">
    Mostrando {page * rowsPerPage + 1} a{' '}
    {Math.min((page + 1) * rowsPerPage, data?.pagination.total ?? 0)} de{' '}
    {data?.pagination.total ?? 0} resultados
  </div>
  <Pagination
    count={totalPages}
    page={page + 1}
    onChange={(newPage) => setPage(newPage - 1)}
    size="md"
    color="primary"
  />
</div>;
```

## Tamanhos

```tsx
<Pagination count={10} size="sm" />
<Pagination count={10} size="md" />

```

## Variantes

```tsx
// Com primeira/última página
<Pagination
  count={10}
  page={page}
  onChange={setPage}
  showFirstButton
  showLastButton
/>

// Compacto
<Pagination
  count={10}
  page={page}
  onChange={setPage}
  siblingCount={0}
  boundaryCount={1}
/>
```

## API

| Prop              | Tipo                       | Padrão      | Descrição                    |
| ----------------- | -------------------------- | ----------- | ---------------------------- |
| `count`           | `number`                   | -           | Total de páginas             |
| `page`            | `number`                   | `1`         | Página atual (1-indexed)     |
| `onChange`        | `(page: number) => void`   | -           | Callback quando muda         |
| `size`            | `'sm' \| 'md' \| 'lg'`     | `'md'`      | Tamanho                      |
| `color`           | `'primary' \| 'secondary'` | `'primary'` | Cor                          |
| `showFirstButton` | `boolean`                  | `false`     | Mostra botão primeira página |
| `showLastButton`  | `boolean`                  | `false`     | Mostra botão última página   |
