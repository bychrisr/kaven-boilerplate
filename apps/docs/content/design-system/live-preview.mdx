# Live Preview

> O componente `<ComponentPreview>` é usado para renderizar exemplos visuais interativos na documentação.

## O que é

O `ComponentPreview` é um wrapper que cria um container isolado com fundo estilizado, borda e padding adequado para exibir componentes UI. Ele simula como o componente aparecerá em uma aplicação real.

## Como Usar

Para usar o Live Preview, importe o componente e envolva seu exemplo JSX nele.

```tsx
import { ComponentPreview } from '@/components/ComponentPreview';
import { Button } from '@/components/ui/button';

<ComponentPreview>
  <Button>Meu Botão</Button>
</ComponentPreview>;
```

### Resultado

<ComponentPreview>
  <Button>Meu Botão</Button>
</ComponentPreview>

## Convenção de Código

Geralmente, acompanhamos o preview visual com o bloco de código correspondente abaixo dele:

<ComponentPreview>
  <div className="flex gap-2">
    <Button variant="contained">Salvar</Button>
    <Button variant="outlined">Cancelar</Button>
  </div>
</ComponentPreview>

```tsx
<div className="flex gap-2">
  <Button variant="contained">Salvar</Button>
  <Button variant="outlined">Cancelar</Button>
</div>
```

Para exemplos que precisam de estado (como abrir um modal, trocar tabs, ou inputs controlados), **você deve criar um componente separado** e importá-lo. O MDX do Next.js App Router não suporta funções inline como children.

1. Crie o componente em `apps/docs/components/examples/MeuExemplo.tsx` com `'use client'`.
2. Importe e use dentro do `ComponentPreview`.

```tsx
// apps/docs/components/examples/LivePreviewInteractiveDemo.tsx
'use client';

import * as React from 'react';
import { Button } from '@/components/ui/button';

export function LivePreviewInteractiveDemo() {
  const [count, setCount] = React.useState(0);
  return (
    <Button onClick={() => setCount((c) => c + 1)}>Clicks: {count}</Button>
  );
}
```

```mdx
import { LivePreviewInteractiveDemo } from '@/components/examples/LivePreviewInteractiveDemo';

<ComponentPreview>
  <LivePreviewInteractiveDemo />
</ComponentPreview>
```

### Resultado

<ComponentPreview>
  <LivePreviewInteractiveDemo />
</ComponentPreview>

```tsx
const [count, setCount] = React.useState(0);

<Button onClick={() => setCount((c) => c + 1)}>Clicks: {count}</Button>;
```

## Props

| Prop        | Tipo                           | Padrão | Descrição                                        |
| ----------- | ------------------------------ | ------ | ------------------------------------------------ |
| `children`  | `ReactNode`                    | -      | O componente a ser renderizado.                  |
| `className` | `string`                       | `''`   | Classes CSS adicionais para o container externo. |
