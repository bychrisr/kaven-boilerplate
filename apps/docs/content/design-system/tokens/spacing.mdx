# Espaçamento

Sistema de espaçamento baseado em múltiplos de 8px para consistência e harmonia visual.

<Callout type="info">
  Todos os espaçamentos seguem a escala de 8px para manter consistência visual.
</Callout>

## Escala de Espaçamento

Base: **8px**

| Valor | Pixels | Classe Tailwind | Uso             |
| ----- | ------ | --------------- | --------------- |
| 0     | 0px    | `p-0`, `m-0`    | Sem espaçamento |
| 1     | 8px    | `p-2`, `m-2`    | Mínimo          |
| 2     | 16px   | `p-4`, `m-4`    | Pequeno         |
| 3     | 24px   | `p-6`, `m-6`    | Médio           |
| 4     | 32px   | `p-8`, `m-8`    | Grande          |
| 6     | 48px   | `p-12`, `m-12`  | Extra grande    |
| 8     | 64px   | `p-16`, `m-16`  | Seções          |
| 12    | 96px   | `p-24`, `m-24`  | Grandes seções  |

## Padding

### Componentes

```tsx
// Card - 24px
<Card className="p-6">
  Conteúdo
</Card>

// Button - 12px 24px
<Button className="px-6 py-3">
  Texto
</Button>

// Input - 16px
<Input className="p-4" />
```

### Containers

```tsx
// Container principal - 24px
<main className="p-6">
  Conteúdo da página
</main>

// Section - padding vertical 48px
<section className="py-12">
  Seção
</section>
```

## Margin

### Espaçamento entre Elementos

```tsx
// Stack vertical - gap 16px
<div className="space-y-4">
  <Card>Item 1</Card>
  <Card>Item 2</Card>
</div>

// Stack horizontal - gap 24px
<div className="flex gap-6">
  <Button>Ação 1</Button>
  <Button>Ação 2</Button>
</div>
```

## Grid System

```tsx
// Grid 3 colunas com gap 24px
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  <Card>Item 1</Card>
  <Card>Item 2</Card>
  <Card>Item 3</Card>
</div>
```

## Border Radius

| Classe         | Valor  | Uso                    |
| -------------- | ------ | ---------------------- |
| `rounded-sm`   | 4px    | Inputs, badges         |
| `rounded`      | 8px    | Botões, cards pequenos |
| `rounded-lg`   | 12px   | Cards, modais          |
| `rounded-xl`   | 16px   | Cards destacados       |
| `rounded-2xl`  | 24px   | Hero sections          |
| `rounded-full` | 9999px | Avatares, pills        |

## Boas Práticas

- Use múltiplos de 8px sempre que possível
- Mantenha consistência em componentes similares
- Use `gap` ao invés de margin em flexbox/grid
- Evite valores arbitrários (`p-[13px]`)
