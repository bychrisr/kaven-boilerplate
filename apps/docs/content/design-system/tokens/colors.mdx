# Cores

O sistema de cores do Kaven Platform é baseado em paletas cuidadosamente selecionadas para garantir acessibilidade, consistência e uma experiência visual agradável.

<Callout type="info">
  Todas as cores seguem as diretrizes WCAG 2.1 AA para contraste e
  acessibilidade.
</Callout>

## Uso e Boas Práticas

<Callout type="warning">
  Sempre teste o contraste de cores para garantir acessibilidade. Use
  ferramentas como o [WebAIM Contrast
  Checker](https://webaim.org/resources/contrastchecker/).
</Callout>

### Hierarquia de Cores

| Uso              | Cor Recomendada                    |
| ---------------- | ---------------------------------- |
| Ação primária    | Primary main                       |
| Ação secundária  | Secondary main ou Primary light    |
| Sucesso          | Success main                       |
| Aviso            | Warning main                       |
| Erro             | Error main                         |
| Informação       | Info main                          |
| Texto principal  | Gray 900 (light) / Gray 100 (dark) |
| Texto secundário | Gray 600 (light) / Gray 400 (dark) |
| Bordas           | Gray 200 (light) / Gray 700 (dark) |
| Backgrounds      | Gray 50 (light) / Gray 900 (dark)  |

### Contraste Mínimo (WCAG 2.1 AA)

- **Texto normal**: 4.5:1
- **Texto grande (18px+)**: 3:1
- **Elementos UI**: 3:1
