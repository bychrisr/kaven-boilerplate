# Tipografia

Utilizamos duas famílias de fontes principais:

- **Public Sans / DM Sans**: Para interfaces e textos (UI).
- **Barlow**: Para títulos e displays.

## Escala

| Classe      | Tamanho | Line Height | Uso              |
| :---------- | :------ | :---------- | :--------------- |
| `text-xs`   | 12px    | 18px        | Labels, badges   |
| `text-sm`   | 14px    | 20px        | Corpo secundário |
| `text-base` | 16px    | 24px        | Corpo principal  |
| `text-lg`   | 18px    | 28px        | Destaques        |
| `text-xl`   | 20px    | 28px        | Títulos H4       |
| `text-2xl`  | 24px    | 32px        | Títulos H3       |
| `text-3xl`  | 30px    | 36px        | Títulos H2       |
| `text-4xl`  | 36px    | 40px        | Títulos H1       |

## Pesos

- **Regular (400)**: Texto comum
- **Medium (500)**: UI Elements
- **SemiBold (600)**: Highlights
- **Bold (700)**: Títulos
