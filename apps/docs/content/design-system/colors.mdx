# Cores

Sistema de cores 5-tone inspirado no Minimals.cc.

## Paleta Semântica

### Primary (Blue)

O sistema de cores do Kaven Platform é baseado em paletas cuidadosamente selecionadas para garantir acessibilidade, consistência e uma experiência visual agradável.

<Callout type="info">
  Todas as cores seguem as diretrizes WCAG 2.1 para contraste e acessibilidade.
</Callout>

## Paleta Primária

A cor primária é usada para ações principais, links e elementos de destaque.
