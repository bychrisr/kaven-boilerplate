# Design System Kaven

Bem-vindo à documentação do Design System da Kaven.

Esta documentação serve como a fonte única de verdade para construção de interfaces na nossa plataforma.

## Princípios

- **Consistência**: Componentes e padrões unificados.
- **Performance**: Otimizado para Next.js e React.
- **Acessibilidade**: Foco em WCAG AA.
