# CSS Variables

> Sistema completo de variáveis CSS que alimenta o Kaven UI v2, incluindo glassmorphism adaptativo, tipografia semântica e cores OKLCH.

## Visão Geral

O Kaven UI v2 utiliza um sistema robusto de variáveis CSS que permite:

- ✅ Temas adaptativos (Light/Dark)
- ✅ Glassmorphism configurável
- ✅ Tipografia semântica
- ✅ Cores em OKLCH (alta fidelidade)
- ✅ Customização centralizada

**Localização:** `apps/admin/app/globals.css`

---

## Glassmorphism Variables

### Light Mode (Frosted Ice)

```css
:root {
  /* Gradientes de fundo */
  --glass-gradient-start: rgba(255, 255, 255, 0.8);
  --glass-gradient-end: rgba(255, 255, 255, 0.6);

  /* Bordas e highlights */
  --color-glass-border: rgba(255, 255, 255, 0.5);
  --color-glass-highlight: rgba(255, 255, 255, 0.8);

  /* Sombras */
  --glass-shadow: 0 8px 30px rgba(0, 0, 0, 0.04);

  /* Spotlight effect */
  --spotlight-color: rgba(255, 255, 255, 0.5);
}
```

### Dark Mode (Deep Tech)

```css
.dark {
  /* Gradientes de fundo */
  --glass-gradient-start: rgba(17, 25, 40, 0.75);
  --glass-gradient-end: rgba(17, 25, 40, 0.5);

  /* Bordas e highlights */
  --color-glass-border: rgba(255, 255, 255, 0.08);
  --color-glass-highlight: rgba(255, 255, 255, 0.05);

  /* Sombras */
  --glass-shadow: 0 8px 32px rgba(0, 0, 0, 0.4);

  /* Spotlight effect */
  --spotlight-color: rgba(255, 255, 255, 0.03);
}
```

### Uso Prático

```css
/* Aplicar glassmorphism em um elemento */
.glass-panel {
  background: linear-gradient(
    135deg,
    var(--glass-gradient-start),
    var(--glass-gradient-end)
  );
  backdrop-filter: blur(10px) saturate(180%);
  border: 1px solid var(--color-glass-border);
  box-shadow: var(--glass-shadow);
}
```

## Typography Variables

### Font Families

```css
:root {
  /* Display - Títulos e headers */
  --font-display: 'Plus Jakarta Sans', sans-serif;

  /* Body - Texto corrido */
  --font-body: 'Inter', sans-serif;

  /* Mono - Código e números */
  --font-mono: 'Source Code Pro', monospace;
}
```

### Uso em Tailwind

```tsx
// Display (títulos)
<h1 className="font-display font-bold">Título Principal</h1>

// Sans (texto)
<p className="font-sans">Parágrafo de texto corrido</p>

// Mono (código/números)
<code className="font-mono">const value = 123;</code>
```

### Mapeamento Tailwind

```css
@theme inline {
  --font-display: var(--font-display);
  --font-sans: var(--font-body);
  --font-mono: var(--font-mono);
}
```

```css
  --radius-sm: calc(var(--radius) - 4px); /* 6px */
  --radius-md: calc(var(--radius) - 2px); /* 8px */

  --radius-3xl: calc(var(--radius) + 12px); /* 22px */
  --radius-4xl: calc(var(--radius) + 16px); /* 26px */
```

## Customização

### Sobrescrever Variáveis

```css

:root {
  /* Mudar cor primária */
  --primary: oklch(0.5 0.2 200); /* Azul customizado */

  /* Ajustar glassmorphism */
  --glass-gradient-start: rgba(255, 255, 255, 0.9);
  --glass-shadow: 0 12px 40px rgba(0, 0, 0, 0.06);

  /* Customizar tipografia */

}
```

## Troubleshooting

### Variável não funciona

**Problema:** Variável CSS não está sendo aplicada.

**Solução:**

```css
/* ❌ Errado - sem var() */
.element {
  color: --primary;
}

/* ✅ Correto - com var() */
.element {
  color: var(--primary);
}
```

### Glassmorphism não aparece

**Problema:** Efeito glassmorphism não visível.

**Solução:**

```css
/* Certifique-se de incluir backdrop-filter */
.glass-panel {
  background: linear-gradient(
    135deg,
    var(--glass-gradient-start),
    var(--glass-gradient-end)
  );
  backdrop-filter: blur(10px) saturate(180%);
  -webkit-backdrop-filter: blur(10px) saturate(180%); /* Safari */
}
```

### Cores diferentes em Light/Dark

**Problema:** Cores não adaptam entre temas.

**Solução:**

```tsx
/* ❌ Errado - cor hardcoded */
<div className="bg-white text-black">

/* ✅ Correto - usa variáveis */
<div className="bg-background text-foreground">
```
