# Shadows

> Sombras para criar hierarquia e profundidade visual.

## Níveis de Elevação

| Classe        | Sombra       | Uso              |
| ------------- | ------------ | ---------------- |
| `shadow-none` | Nenhuma      | Elementos planos |
| `shadow-sm`   | Sutil        | Cards sutis      |
| `shadow`      | Padrão       | Cards padrão     |
| `shadow-md`   | Média        | Dropdowns        |
| `shadow-lg`   | Grande       | Modais           |
| `shadow-xl`   | Extra grande | Popovers         |
| `shadow-2xl`  | Enorme       | Overlays         |

## Exemplos

```tsx
// Card com sombra padrão
<div className="bg-card rounded-lg shadow p-6">
  Card com sombra
</div>

// Dropdown com sombra média
<div className="bg-popover rounded-lg shadow-md p-4">
  Dropdown
</div>

// Modal com sombra grande
<div className="bg-card rounded-lg shadow-lg p-6">
  Modal
</div>
```
