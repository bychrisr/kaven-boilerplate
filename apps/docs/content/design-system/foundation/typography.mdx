# Typography

> Sistema tipográfico completo com DM Sans e Barlow para criar hierarquia visual clara.

## Famílias de Fontes

### DM Sans (Principal)

Fonte sans-serif moderna para UI e conteúdo.

**Pesos:** Regular (400), Medium (500), Semi Bold (600), Bold (700)

**Uso:** Corpo de texto, botões, formulários, navegação

<div className="p-6 bg-white dark:bg-gray-800 rounded-lg border my-4 space-y-2">
  <div className="font-sans text-base">Texto em DM Sans Regular</div>
  <div className="font-sans text-base font-medium">Texto em DM Sans Medium</div>
  <div className="font-sans text-base font-semibold">
    Texto em DM Sans Semi Bold
  </div>
  <div className="font-sans text-base font-bold">Texto em DM Sans Bold</div>
</div>

### Barlow (Display)

Fonte geométrica para títulos e destaques.

**Pesos:** Medium (500), Semi Bold (600), Bold (700)

**Uso:** Títulos principais, logotipos, CTAs

<div className="p-6 bg-white dark:bg-gray-800 rounded-lg border my-4">
  <h1 className="font-display text-4xl font-bold">Título em Barlow</h1>
</div>

## Escala Tipográfica

| Classe      | Tamanho | Line Height | Uso                |
| ----------- | ------- | ----------- | ------------------ |
| `text-xs`   | 12px    | 18px        | Labels, badges     |
| `text-sm`   | 14px    | 20px        | Texto secundário   |
| `text-base` | 16px    | 24px        | Corpo padrão       |
| `text-lg`   | 18px    | 28px        | Texto destacado    |
| `text-xl`   | 20px    | 28px        | Subtítulos         |
| `text-2xl`  | 24px    | 32px        | Títulos de seção   |
| `text-3xl`  | 30px    | 36px        | Títulos de página  |
| `text-4xl`  | 36px    | 40px        | Títulos principais |
| `text-5xl`  | 48px    | 1           | Hero titles        |

## Hierarquia de Títulos

```tsx
// h1 - Título principal
<h1 className="font-display text-4xl font-bold text-gray-900">
  Título Principal
</h1>

// h2 - Seções principais
<h2 className="font-display text-3xl font-semibold text-gray-800">
  Seção Principal
</h2>

// h3 - Subseções
<h3 className="font-sans text-2xl font-semibold text-gray-800">
  Subseção
</h3>
```
