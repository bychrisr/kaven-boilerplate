# Componentes Built-in

Esta página demonstra os componentes built-in do Nextra.

## Callout

import { Callout } from 'nextra/components';

<Callout type="info">
  Informação útil que os usuários devem saber, mesmo ao escanear o conteúdo
  rapidamente.
</Callout>

<Callout type="warning">
  Informação urgente que precisa de atenção imediata do usuário para evitar
  problemas.
</Callout>

<Callout type="error">
  Aviso sobre riscos ou resultados negativos de certas ações.
</Callout>
