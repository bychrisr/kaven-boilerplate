| Token | Valor   | Pixels |
| :---- | :------ | :----- |
| `0`   | 0       | 0px    |
| `1`   | 0.25rem | 4px    |
| `2`   | 0.5rem  | 8px    |
| `3`   | 0.75rem | 12px   |
| `4`   | 1rem    | 16px   |
| `6`   | 1.5rem  | 24px   |
| `8`   | 2rem    | 32px   |
| `12`  | 3rem    | 48px   |
| `16`  | 4rem    | 64px   |
