# Kaven Tenant App

This is the dedicated Tenant Application for Kaven SaaS.
It is a separate Next.js application running on port **3001**.

## Features

- **Space Awareness**: Navigation and data are scoped to the active Space.
- **Team Management**: Self-service team invites and role management.
- **Project & Tasks**: Demo modules for productivity.
- **Settings**: Tenant-scoped configuration.

## Development

```bash
pnpm dev
# Runs on http://localhost:3001
```
